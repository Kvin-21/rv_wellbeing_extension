# RV Wellbeing Extension

A Chrome extension to check in with how RV students are feeling each day - all completely anonymously.

---

## What it does

- **Daily mood check-in** - pick how you're feeling from 5 emotions: Happy, Stressed, Lonely, Excited, or Frustrated
- **Personalised responses** - after selecting a mood, you get a short validation and a practical micro-action depending on the time of day
- **Streak tracking** - keeps a running streak of how many days in a row you've checked in
- **Daily reminder** - An optional notification sent everyday that opens the extension when clicked
- **Three visual themes** - Glass (default), Light, and Dark
- **Feedback tab** - To allow students to submit any thoughts or suggestions directly

---

## Try it out

### Option A - Download as ZIP

1. Click the green **Code** button → **Download ZIP**
2. Once downloaded, unzip the folder somewhere on your computer (e.g. your Desktop)
3. Open Chrome and go to `chrome://extensions`
4. Turn on **Developer mode** using the toggle in the top-right corner
5. Click **Load unpacked**
6. Select the unzipped folder (the one that contains `manifest.json`)
7. The extension icon should appear in your Chrome toolbar, click it to open!

### Option B — Clone with Git

```bash
git clone https://github.com/Kvin-21/rv_wellbeing_extension.git
```

Then follow steps 4–8 above, selecting the cloned `rv_wellbeing_extension` folder when prompted.

---

## Using the extension

1. Click the extension icon in your Chrome toolbar
2. On the **Home** tab, tap how you're feeling today
3. Read your personalised response and try the suggested micro-action
4. Head to **Settings** to enable daily reminders or switch themes
5. Use the **Feedback** tab to share any thoughts!

The streak counter in the top-right shows how many days in a row you've checked in. If you miss a day it resets, so try to keep it going!
